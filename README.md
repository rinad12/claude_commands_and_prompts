# Claude Commands & Prompts 🤖

This is a personal collection of my custom commands and prompts designed to optimize interactions with Claude AI. 

> **Note:** I created this repository primarily for my own use, but feel free to explore and use anything you find helpful!

## 📂 Repository Structure

The project is organized into two main directories:

* **`/commands`** – Short, functional instructions, "slash commands," and technical rules to quickly adjust the model's behavior or output format.
* **`/prompts`** – Detailed, long-form prompts, templates for complex tasks, chain-of-thought instructions, and specific personas.

---

## 🚀 How to Use

1.  Navigate to the desired folder.
2.  Open the Markdown file and copy the content.
3.  Paste it into the .claude folder
